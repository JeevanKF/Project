# Project: 

# Predictive Maintenance of Industrial Machines (LSTM vs Linear Regression)

## Overview
This project predicts Remaining Useful Life (RUL) of industrial machines using Machine Learning models.

It compares:
- Linear Regression (baseline)
- LSTM (deep learning model)

Goal: Detect early degradation and prevent critical machine failure.

---

## Approach

- Data preprocessing and cleaning
- Feature scaling (MinMaxScaler)
- RUL calculation and capping
- Model training:
  - Linear Regression
  - LSTM (sequence-based learning)
- Evaluation using:
  - RMSE
  - MAE
  - R² Score

---

## Key Results

- LSTM outperformed Linear Regression
- Lower RMSE & MAE
- Higher R² score → better prediction accuracy
